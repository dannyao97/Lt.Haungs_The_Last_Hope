Lt. Haungs - The Last Hope

Author: Daniel Yao

To play the game, download the .jar file and either:
-Double-click on the downloaded file.
or
-Open a terminal and navigate to the directory where the .jar file was downloaded
-Run the command*
	"java -jar Lt.Haungs_TLH.jar"

*NOTE If the command does not work, check to make sure Java is installed by using the command
	"java -version"
